# GiftOS Core — Code of Conduct

## Our Pledge

We pledge to make GiftOS Core a professional, inclusive, and financially responsible open-source project.

## Standards

### Financial Integrity
- All financial calculations must be auditable
- Exchange rate logic must be transparent
- Transaction handling must be secure
- No hidden fees or undisclosed monetization

### Open-Core Governance
- Core features remain open source
- Commercial extensions clearly separated
- Fair contributor recognition
- Transparent roadmap and decision-making

### Unacceptable
- Financial data manipulation
- Security vulnerability exploitation
- Harassment or discrimination
- Plagiarism of trading strategies
